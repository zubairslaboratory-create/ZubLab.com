Registration Policy

To ensure fair access for all learners, ZubLab has implemented the following registration rules:

One Registration Per Device

Each device can only register one account. This prevents multiple registrations from the same device.

Unique Email Address

Each email address can only be used for one registration. Duplicate email registrations are not allowed.

How It Works

Our system detects if your device has already registered or if your email is already in use. If you've already registered, you'll see a confirmation message instead of the registration form.
